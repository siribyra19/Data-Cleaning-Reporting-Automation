# Data Cleaning & Reporting Automation

## Project Overview

This project automates the process of cleaning raw datasets and generating reports using Python.

## Features

- Handles missing values
- Removes duplicate records
- Cleans inconsistent text values
- Generates summary statistics
- Creates visual reports
- Exports cleaned dataset automatically

## Tools Used

- Python
- Google Colab
- Pandas
- Matplotlib

## Output Files

- cleaned_data.csv
- summary_report.csv
- sales_chart.png

## Conclusion

The project demonstrates automated data preprocessing and reporting, reducing manual effort and improving data quality.
